# Admin Cliente

Frontend Angular 13
Backend PHP

## CONFIGURAÇÃO DOS AMBIENTES

### AMBIENTE EM CONTAINERS DOCKER

#### REQUISITOS

- Ter instalado docker e docker-compose
#### Run

Para iniciar o ambiente:

- Na raiz do projeto, executar.

    ```
    $ docker-compose up
    ```
- Frontend
     ```
    $ http://localhost:4200
    ```
- Backend
     ```
    $ http://localhost:8000
    ```
